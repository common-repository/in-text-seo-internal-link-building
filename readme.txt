== Installation ==

To set up the Free plugin:

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Go to the Plugins page and activate the plugin creating a user on the NotedLinks service (It's free)
3. Use the Settings page to set up your general links settings.
4. Go to the dashboard to set up the link rules for you website.
5. That's all. You can check your posts for new links.
